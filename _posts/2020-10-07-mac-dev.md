## Homebrew
### install homebrew
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
### add path to .zshrc
terminal에서 해당 폴더 찾아서 확인할 것
```
# for arm chip
export HOMEBREW_HOME="/opt/homebrew"
# for intel chip
#export HOMEBREW_HOME="/usr/local/Homebrew"
export PATH="$HOMEBREW_HOME/bin:$PATH"
export PATH="$HOMEBREW_HOME/sbin:$PATH"
```

## Rosetta 2
- Intel chip용으로 개발된 App을 Arm chip (M1, M2...) Mac에서 실행하기 위함
- 구형 App을 구동하기 위함이 아니면 설치 의미 없을 듯 (Pytorch 초기 버전에서 M1 chip을 지원하기 위해 Rosetta 설치가 필요한 적이 있었음)
```
sudo softwareupdate --install-rosetta
```

### AdoptOpenJdk: Deprecated
### Install Temurin (https://adoptium.net/installation/)
```
brew install --cask temurin@8
brew install --cask temurin@11
brew install --cask temurin@17
brew install --cask temurin@21
brew install --cask temurin@25
```
### jenv (java version switching tool)
#### install
```
brew install jenv
```
update .zshrc
```
# jenv config
export PATH="$HOME/.jenv/bin:$PATH"
eval "$(jenv init -)"
```
#### register jdk path to jenv
```
jenv add /Library/Java/JavaVirtualMachines/temurin-8.jdk/Contents/Home
jenv add /Library/Java/JavaVirtualMachines/temurin-11.jdk/Contents/Home
jenv add /Library/Java/JavaVirtualMachines/temurin-17.jdk/Contents/Home
jenv add /Library/Java/JavaVirtualMachines/temurin-21.jdk/Contents/Home
jenv add /Library/Java/JavaVirtualMachines/temurin-25.jdk/Contents/Home
```
#### set default java version
```
jenv global temurin64-17.0.9
```
#### enable jenv plugin
```
jenv enable-plugin export
```

#### usage
create env
```
conda create -n {env-name} python={version}
```
remove env
```
conda env remove -n {env-name}
```
search env list
```
conda env list
```
activate/deactivate
```
conda activate {env-name}
conda deactivate
```
### PyCharm에서 활용
