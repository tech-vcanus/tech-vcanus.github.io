# Install Development Environment (Windows10)

## Install wsl2
 - link: https://docs.microsoft.com/en-us/windows/wsl/install
 - execute Powershell with admin
 ```
 $ wsl --install
 ```
 - check wsl version (Docker needs wsl2)
 ```
 $ wsl -l -v
 PS C:\Windows\system32> wsl -l -v
  NAME      STATE           VERSION
* Ubuntu    Running         2
 ```
