# IOS 개발 환경
## Java, Python 등 기본 환경 설치
## XCode 설치
- 앱 다운로드 및 설치
- Command Line Tool 설치
```
$ xcode-select --install
```
## Rosetta 2 설치
```
$ sudo softwareupdate --install-rosetta --agree-to-license
```
## CocoaPods 설치
- Arm Chip 노트북인 경우 gem install cocoapods에 문제가 있음
- brew로 설치할 것을 권장
```
$ brew update
$ brew install cocoapods
```
